Plugins

Each plugin consists of three elements:

  • name
  • options
  • methods



Basic plugin syntax.

Note: The following code is executed, so dont forget to check the console. 

$.plug('play',{ // plugin name
  defaults: {
      option: 'this is the default option'
  },
  init: function () { // first function that gets called
    this.el,  // javascript object
    this.$el, // jQuery object
    this.opts // options merged from defaults, passed options and data attributes

    this.method(); // calling a method
  },
  method: function (el,$el,options) { // custom method
    console.log('play.method has been called')
    console.log(this.opts.option)
    console.log('\n')
  }
});


Calling the plugin

We can call the plugin in different ways:


This is a basic plugin call without any options.
It will invoke the init function and nothing more.


$(function () {
  // will call init and then play.method
  // output will be the logs from play.method
  $('.yourElement').play();
})

In this example we've overwritten the defaults:


$(function () {
  // passing options which will overwrite the defaults if set
  $('.yourElement').play({
    option: 'overwriting defaults'
  });
})

Here we added a callback:


$(function () {
  // if first attribute is a function it will get executed after everything else is finished
  $('.yourElement').play(function(){
    console.log('this is a callback')
    console.log('\n')
  });
})

Mixing callbacks and default overwrites:


$(function () {
  // mixing options overwrite and callbacks
  $('.yourElement').play({
    option: 'overwriting defaults and callbacks'
  },function(){
    console.log('this just another callback')
    console.log('\n')
  });

})

For better usability there as another option to overwriting the defaults. All data attributes with a specific syntax will do this:


$(function () {

  // lets set a data attribute since they can also overwrite the defaults 
  $('.yourElement').attr('data-play-option','data attribute overwrite');
  // Note: The syntax for this is data-pluginname-optionname

  $('.yourElement').play();  

})
