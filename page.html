Arguments are the same as with a plugin call:


$(function () {

  // You can pass the same arguments on a method which you can on its plugin
  $('.yourSecondElement').play('method',{
    otpion: 'method option overwrite'
  },function(){
    console.log('method callback')
  })

})

Modifying

You can overwrite the function of a plugin method. This gives you great controll over external plugins written with plug.js.

For this task we use the child function overwrite from $.plug


$(function () {

  $.plug.replace('play','method',function () {
    console.log('this is an overwrite');
    // note that the option is still the data attribute of our element
    console.log('Option log after overwrite: ',this.opts.option)
  });

})
